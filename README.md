# ポートフォリオサイト

[Next.js](https://nextjs.org) で構築したポートフォリオサイトです。

## 開発環境の起動

開発サーバーを起動するには、以下のコマンドを実行してください：

```bash
npm run dev
# または
yarn dev
# または
pnpm dev
# または
bun dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くと確認できます。

`app/page.tsx` を編集すると、ページが自動的に更新されます。

このプロジェクトでは [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) を使用して、[Geist](https://vercel.com/font) フォントを自動的に最適化・読み込みしています。
